Message par Mensha » 06 mars 2023, 14:09

Apparemment un problème de ram https://softwarekeep.com/blog/what-are- ... ntify-them

Débrancher et rebrancher la ram, si ca continue, tenter de booter la cm sans la ram : Normalement le bios devrait au moins apparaître vue que tu as les beep code qui fonctionnent encore. Si t'as le bios la ram est morte.

La cm commence a dater, si la ram est morte, regarder si les condensateurs sont pas gonflé éclater, si c'est le cas faut envisager de la changer aussi, voir le proc si tu trouves pas une cm pas cher.

Edit : s'il y a plusieurs barrettes de ram, les rebancher une par une, avec de la chatte y en juste une de cramée
