L'erreur serait sans doute d'adapter une storyline déjà existante. Mieux vaut qu'ils partent sur une random aventure avec un nobody qui se retrouve embarqué dans une aventure plus grande que lui, impliquant des Arpenteurs, jusqu'à éveiller sa propre Étincelle.

Enfin perso je pense que le bon format pour aborder MtG n'est pas un film live action mais une série animée.

Message par Hermelk » 17 mai 2023, 09:04

Pareil que Kaptain et Merk et Pazu, vu avant-hier, c'était trop bien.

Le scénar est simple mais efficace. Regé-Jean est génial, super pirouette post-Bridgerton, mais tous les acteurs sont excellents (petit bémol sur le mage que j'ai trouvé légèrement en-dessous par moment, mais c'est mineur).

Les différentes mini-scènes sont à la fois bien faites d'un point de vue écriture, et d'un point de vue graphique (c'est magnifique je trouve, à tous les niveaux).
Les tomates mises avec les aromates se parfument et évitent partiellement les limaces. Ce qui n'empêche pas les règles, pas d'ail avec les tomates; les déchets de table c'est très bien au compost mais pas sous le lit.
