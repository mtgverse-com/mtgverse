Exemple de fiche :

Groupe : Les méchants / Alignement : Villageois / Points de dégats : 0


ghost.JPG (33.89 Kio) Consulté 323 fois
GHOST RIDER : (Actif, unique, à tout moment) : Cible un joueur mort, il revient dans la game.

Punisher.JPG (36.51 Kio) Consulté 323 fois
PUNISHER : (Actif, une fois par nuit) : Cible un joueur, il prend un point de dégât.

adam.JPG (38.02 Kio) Consulté 323 fois
ADAM WARLOCK : (Actif, unique, en journée) : Tous les héros de cout 3 ne peuvent etre activés/déclenchés pour cette journée.

Lexique :
** Fiche : Nom du groupe / Alignement / points de dégats au début de la partie.
** Points de dégats : quand vous arrivez à trois, vous mourez instantanément.
** Unique : une seule utilisation pour toute la game
** Cout d'un héros: C'est le chiffre en bleu sur l'image. Pour le moment, les chiffres en orange n'ont pas d'utilités.
** Sur cette game, héros = pouvoir.

Mécanique Emplacements

Le mardi (~7h) est annoncé 3 emplacements. Gauche, milieu, Droite. Celui de Gauche à son effet révélé, les autres non.

Exemple :

Gauche : La maison du pere Noel : Tout joueur sur cet emplacement recevra une vie supplémentaire
Milieu : La mare au canards : ***
Droite : L'enfer : ***

** Vous avez jusqu'a mardi 21h pour envoyer par mp au MJ votre choix. Soit vous etes audacieux et choisissez un emplacement inconnu, soit vous jouez sécure et choisissez l'emplacement révélé.
** Le Mercredi matin (~7h) sera révélé les effets des deux autres emplacements.
** Mercredi 21h, on résout les effets de tous les emplacements.
